# Sorting & Searching Learning Web App

This is a React + Vite web app.

## How to run

1. Open terminal inside this folder.
2. Install dependencies:

```bash
npm install
```

3. Start the website:

```bash
npm run dev
```

4. Open the local link shown in the terminal, usually:

```text
http://localhost:5173
```

## What is inside

- Bubble Sort explanation
- Insertion Sort explanation
- Merge Sort explanation
- Big O notation using loops only
- Linear Search explanation
- Binary Search explanation
- Interactive visualization GUI
- Delay slider for step-by-step demo
- Bar count slider
- Target value for searching
